样式化 HTML 元素

  • 元素选择器：即直接匹配 HTML 元素的选择器。
    •         h1 {
          color: red; 
        }
  • 可以一次使用多个选择器，只需要使用逗号分隔即可。
    •         h1, li {
          collor: green;
        }

改变元素的默认行为

浏览器会为 HTML 文件添加默认的 CSS 样式。使用下面的代码移除浏览器提供的默认样式。

      li {
        list-style-type: none;
      }

使用类名

为想要使用同一样式的元素添加具有相同值的 class 属性

      <ul>
        <li>项目一
        <li class="special">项目二
        <li>项目 三
      </ul>

在 CSS 里使用 .className 来选中给定的 class。

      .special {
        color: orange;
        font-weight: bold;
      }

元素和类名一起出现时表示选中每个该类的该元素，注意是该类的该元素。

      li.special {
      color: orange;
      font-weight: bold; 
    }

也可以出现多个。

      li.special,
      span.special {
        color: orange;
        font-weight: bold;
      }

根据元素在文档中的位置确定样式

  • 包含选择符
    •         li em { /*设置在 li 中 em 的样式*/
          color: rebeccapurple;
        }
  • 相邻选择符
    •         h1 + p { /*设置与 h1 相邻的 p 的样式*/
          font-size: 200%;
        }

根据状态确定样式

        a:link { /*未访问的链接为粉色*/
          color: pink;
        }

        a:visited { /*已访问的链接为绿色*/
          color: green;
        }

        a:hover { /*悬浮在链接上时 移除链接的下划线*/
          text-decoration: none;
        }

选择器和选择符可以同时使用

      body h1 + p .special {
        color: yellow;
        background-color: black;
        padding: 5px;
      }
