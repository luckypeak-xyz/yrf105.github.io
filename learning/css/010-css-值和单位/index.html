了解 CSS 属性中使用的不同类型的值和单位。

什么是 css 的值？

css 规范和 MDN 的属性页上，值以被尖括号包围的形式出现：<color>。

CSS 中的值是一种定义允许子值集合的方法。这意味着如果您看到 <color> 是有效的，那么不同类型的颜色值—关键字、十六进制值、rgb()函数等等就都是有效的。

数字，长度和百分比

数值包括：integer，number，dimension，percentage。

长度

绝对长度

cm, mm, Q(四分之一毫米), in 英寸，pc 十二点活字，pt 点，px 像素。

相对长度

  • em - 在 font-size 中是相对于父元素字体大小，在其他属性中使用是相对对自身的字体大小
  • ex - 字符 x 的高度
  • ch - 字符 0 的宽度
  • rem - 根元素的字体大小
  • lh - 元素的 line-height
  • vw - 视窗宽度的 1%
  • vh - 视窗高度的 1%
  • vmin - 视窗较小尺寸的 1%
  • vmax - 视窗较大尺寸的 1%

ems 和 rems

  • 1.5em - 是其父嵌套字体大小的 1.5 倍
  • 1.5rm - 是其根元素字体大小的 1.5 倍

百分比

相对父元素相同属性的值来设置。

数字

透明属性 opacity 接收不添加任何单位的数字 [0 - 1]

颜色

现代计算机的标准色系统是 24 位的。

css 指定颜色的方法有：

  • 颜色关键字
  • 十六进制 RGB 值
  • RGB 和 RGBA
  • HSL 和 HSLA

图片

  • url() - 图像链接
  • linear-gradient - 渐变

位置

background-position: x y; 左上角为原点。

字符串和标识符

引号括起来的内容为字符串。

函数

例如：calc() rgb() hsl() url()
