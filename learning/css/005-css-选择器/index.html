css 选择器

目标：详细学习 css 选择器的工作方式。

什么是选择器

选择器是一种告诉浏览器 HTML 中哪个元素被选中应用响应 css 属性值的方式。

被选择的元素叫：选择器的对象

选择器列表

可以将具有相同属性，相同属性值的不同选择器组合为选择器列表：

将

      h1 {
        color: blue;
      }

      .special {
        color: blue;
      }

组合为：

      h1, .special {
        color: blue;
      }

选择器的种类

  • 元素选择器：h1
  • 类选择器：.className
  • id 选择器：#id
  • 标签属性选择器：
    •         a[title] { } /*若 a 有属性 title 则被选中*/
        a[href="https://example.com"] { } /*若 a 的 href 属性的值为 https://example.com 则被选中*/
  • 伪类：a:hover 当 a 的状态为 指针悬浮在其上 时被选中
  • 伪元素：p::first-line 段落的第一行被选中
选择器 示例
类型选择器 h1 {}
通配选择器 *
类选择器 .box {}
id 选择器 #unique {}
标签属性选择器 a[title] {}
伪类选择器 a:hover {}
伪元素选择器 p::first-line {}
后代选择器 article p {}
子代选择器 article > p {}
相邻兄弟选择器 h1 + p {}
通用兄弟选择器 h1 ~ p {}
