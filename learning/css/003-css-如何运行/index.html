浏览器是如何获取 HTML 和 CSS 并将他们加载成网页的呢？

浏览器究竟是如何运行的？

关于 DOM

一个 DOM 有一个树形结构，标记语言中的每一个元素、属性以及每一段文字都对应着结构树中的一个节点（Node/DOM 或 DOM node）。节点由节点本身和其他 DOM 节点的关系定义，有些节点有父节点，有些节点有兄弟节点（同级节点）。

一个真实的 DOM 案例

如下代码：

      <p>
        Let's use:
        <span>Cascading</span>
        <span>Style</span>
        <span>Sheets</span>
      </p>
对应的 DOM 为： 
      P
      ├─ "Let's use:"
      ├─ SPAN
      |  └─ "Cascading"
      ├─ SPAN
      |  └─ "Style"
      └─ SPAN
          └─ "Sheets"

应用 CSS 到 DOM

浏览器会解析 HTML 并创造一个 DOM，然后解析 CSS。把规则应用到对应的元素上并渲染图像到屏幕。
