图像，媒体和表单元素

理解某些元素经 CSS 样式化后的非常规表现。

替换元素

图片，视频，iframe 等称为替换元素，这意味着 CSS 不能影响它们的内部布局——而仅影响它们在页面上相对于其它元素的位置。

调整图像大小

如何处理溢出问题？

  • max-width: 100%;
  • object-fit: [cover contain fill];

布局中的替换元素

flex 或者 grid 布局中，默认情况下元素会被拉伸到充满整块区域。但是图像不会被拉伸，而会对齐到网格区域或者弹性容器的起始处。

强制图片拉伸以充满其所在网格：

      img {
      width: 100%;
      height: 100%;
      }

继承和表单元素

表单元素默认不会继承字体样式，要想让表单元素继承父元素定义的字体需要：

      button,
      input,
      select,
      textarea {
        font-family : inherit;
        font-size : 100%;
      }

其他有用的设置

应该在 textarea 上设置 overflow: auto 以避免 IE 在不需要滚动条的时候显示滚动条：

      textarea {
        overflow: auto;
      }

统一的表单重置

      button,
      input,
      select,
      textarea {
        font-family: inherit;
        font-size: 100%;
        box-sizing: border-box;
        padding: 0; margin: 0;
      }

      textarea {
        overflow: auto;
      }
