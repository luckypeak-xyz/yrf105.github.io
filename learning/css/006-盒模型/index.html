块级盒子（block box）和内联盒子（inline box）

外部显示类型为块级盒子（诸如标题 <h1> 等和段落 <p>）的行为如下：

  • 盒子会在内联的方向上扩展并占据父容器在该方向上的所有可用空间，在绝大数情况下意味着盒子会和父容器一样宽
  • 每个盒子都会换行
  • width 和 height 属性可以发挥作用
  • 内边距（padding）, 外边距（margin）和 边框（border）会将其他元素从当前盒子周围“推开”

外部显示类型为内联盒子（用做链接的 <> 元素、 <span>、 <em> 以及 <strong> 都是默认处于 inline 状态的）的行为如下：

  • 盒子不会产生换行。
  • width 和 height 属性将不起作用。
  • 垂直方向的内边距、外边距以及边框会被应用但是不会把其他处于 inline 状态的盒子推开。
  • 水平方向的内边距、外边距以及边框会被应用且会把其他处于 inline 状态的盒子推开。

外部和内部显示类型

我们可以通过使用类似 flex 的 display 属性值来更改内部显示类型。如果设置 display: flex，在一个元素上，外部显示类型是 block，但是内部显示类型修改为 flex。该盒子的所有直接子元素都会成为 flex 元素。

标准盒子模型

      .box {
      width: 350px;
      height: 150px;
      margin: 25px;
      padding: 25px;
      border: 5px solid black;
      }

如果使用标准模型宽度 = 410px (350 + 25 + 25 + 5 + 5)，高度 = 210px (150 + 25 + 25 + 5 + 5)，padding 加 border 再加 content box。margin 不计入盒子大小。

代替盒模型

使用上面相同的样式得到 (width = 350px, height = 150px)。

浏览器默认使用标准模型，添加以下代码使得所有元素都使用代替盒模型：

      html {
      box-sizing: border-box;
      }
      *, *::before, *::after {
      box-sizing: inherit;
      }
