创建超链接

什么是超链接？

几乎任何网络内容都可以转换为超链接，点击超链接将会使网络浏览器转到另一个网址（URL）。

URL 可以指向网络上保存的任何内容，如果浏览器不知道如何打开，会询问你使用本地的其他程序打开。

URL 和 path

URL 统一资源定位符是一个定义了在网路上的位置的一个字符串。

URL 通过路径查找文件。

  • 指向当前目录中的文件。
  • 指向子目录中的文件。
  • 指向上级目录中的文件。（点击上一条查看。）

文档片段

超链接除了可以链接到文档外，也可以链接到 HTML 文档的特定部分（被称为文档片段）。要做到这一点，你必须首先给要链接到的元素分配一个 id 属性。

绝对 URL 和相对 URL

  • 不管绝对 URL 在哪里使用，总是指向确定的位置。
  • 会因为文件的移动而失效。

链接最佳实践

使用清晰地链接措辞

要：

  • 在链接文本中包含关键词。
  • 创建描述性链接。

不要：

  • 不要重复 URL 作为链接文本的一部分。
  • 不要在链接文本中说“链接”或“链接到”。
  • 不要创建过长的链接标签。
  • 不要让相同的多个副本链接到不同地方。

链接到非 HTML 资源——留下清晰地提示

下载销售报告（PDF, 10MB）

观看视频（将在新标签页中播放，HD 画质）

进入汽车游戏（需要 Flash 插件）

在下载链接中使用 download 属性

当你链接到要下载的资源而不是在浏览器中打开时，你可以使用 download 属性来提供一个默认的保存文件名。

电子邮件链接

    <a
    href="mailto:nowhere@mozilla.org?cc=name2@rapidtables.com&bcc=name3@rapidtables.com&subject=The%20subject%20of%20the%20email&body=The%20body%20of%20the%20email">
    Send mail with cc, bcc, subject and body
    </a>

其中：

  • mailto - 打开邮件客户端
  • subject - 主题
  • cc - 抄送
  • body - 主体

每个字段都必须是 URL 编码的。
